## @flypeng/eslint-config

> Based Airbnb JavaScript Base Style To Extend

```sh
pnpm add @flypeng/eslint-config -D
```

To config in eslint configuration file `.eslintrc.js`

```js
{
    "extends": ["@flypeng/eslint-config"]
}
```
